# CAPEducation_contest
Этот Telegram-бот — моё небольшое портфолио, которое я сделал на языке Python с использованием библиотеки telebot. В нём собрана информация обо мне, моих целях, пути в IT, хобби и проектах.

После запуска команды /start пользователю открывается удобное меню с кнопками. Благодаря этому не нужно запоминать команды — достаточно нажать на нужный раздел. В боте можно узнать немного обо мне, прочитать, почему я решил заниматься программированием, какие цели ставлю перед собой и каких результатов уже смог добиться.

Также в боте есть раздел с моими работами. В нём я рассказал о проектах, которые уже успел создать, и оставил ссылки на GitHub, чтобы любой желающий мог посмотреть исходный код. Отдельной кнопкой можно открыть ссылку на мой основной репозиторий.

Во время разработки я научился работать с обработчиками сообщений, создавать удобные клавиатуры, организовывать код по функциям и делать навигацию внутри Telegram-бота простой и понятной. Этот проект помог мне лучше разобраться в создании ботов и получить практический опыт программирования.
